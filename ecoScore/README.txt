Welcome to ECO SCORE!

What is Eco Scoring?
Eco Scoring assesses the overall sustainability of one's lifestyle. 

Features of Eco Score: 
User Registration System
Login/Logout System
Users are able to contribute to Eco Score through the suggestions box which is linked to my email
Users are able to compare see their Eco Score as well as compare it to other users

The 'login' app takes care of user registration and logging in/logging out
The 'ecoscore' app takes care of all the logic to do with Eco Score
The 'trailAssess' app takes care of the main menu as well as URL redirects in all of the other applications
